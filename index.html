반갑습니다! 사용자 경험을 최우선으로 생각하는 신입 프론트엔드 개발자 김유랑 입니다. HTML, CSS, JavaScript와 React를 활용한 다양한 웹 애플리케이션을 개발하며, 깔끔하고 직관적인 UI를 구현하는 데 집중하고 있습니다. 새로운 기술을 배우고 문제를 해결하는 과정에서 성장하는 것을 즐깁니다. 여러분과 함께 일하며 더 나은 웹을 만들어가고 싶습니다

라이브러리

  • Axios

    서버와의 비동기통신을 하는데 HTTP 요청 및 응답 처리가 간편해 사용했습니다.

  • Gsap

    순차적 제어가 가능해 복잡한 애니메이션을 흐름을 관리하는데 유용했습니다.

  • Tailwind CSS

    html에 코드내에서 어떤 스타일이 적용되는지 직관적으로 파악할 수 있어 빠르고 효율적인 UI개발에 도움이 되었습니다.

  • Jquery

    간결한 문법으로 DOM 조작이 간단하고, 다양한 웹 브라우저와의 호환성이 높아 사용했습니다.

  • Swiper.js

    다양한 슬라이드 구현과 유연한 커스터마이징을 할 수 있어 사용했습니다.

프론트

  • React

    가상 DOM구조를 이해하고 컴포넌트 기반구조로 코드의 재사용성과 유지보수가 용이했고, hook을 사용해 상태관리를 간편히 할 수 있었습니다.

  • Redux

    중앙집중식 상태관리를 통해 복잡한 데이터흐름을 예측하는데 도움이 되었고, 미들웨어를 활용해 비동기 작업을 간편히 처리할 수 있었습니다.

  • Vue.js

    반응형 데이터 바인딩과 직관적이고 유연한구조의 프레임워크로 접근성이 좋았습니다.

  • Vite

    빠른 빌드 속도와 즉각적인 업데이트가 유용해 사용했습니다.

  • netlify

백엔드

  • Node.js

    I/O 작업을 비동기적으로 처리하는 서버를 구축해보며 높은 서버응답성능을 경험했고, npm을 통해해 다양한 패키지를 쉽게 사용할 수 있었습니다.

  • Express.js

    최소한의 설정만으로 서버를 빠르게 구축하고, 다양한 미들웨어를 사용해 API서버개발을 학습하는데 도움이 되었습니다.

  • MongoDB

    데이터구조에 제약이 없고 스키마가 유연해 다양한 형태의 데이터를 저장하고 처리할 수 있었습니다.

디자인

  • Figma

    반복되는 컴포넌트디자인의 재사용성이 용이하고, 디자인과 개발간의 일관성이 높아 효율적이었습니다.

  • Photoshop

    고급 기능을 활용해 정밀한 디자인 작업에 유용했습니다.

  • Illustrator

    아이콘, 로고, UI 디자인의 벡터기반 디자인을 생성하는데 최적화되어 사용했습니다.
